# 增加自定义组件

### 在增加自定义组件之前需要了解平台和文件的概念

目前，内置的平台为`pc` 和 `mobile`, 文件为 `index` 和 `view`
主要利用了dynamic import做的动态引入，在切换平台或文件时加载不同的组件
所以在自定义组件时要注意文件的摆放位置


1. 定义组件说明
    在项目`src`目录下创建 `rf-render-components`文件夹

2. 创建平台目录和文件目录结构大致如下
```
|--rf-render-components  
    |--pc  
        |--Button
            |--index.tsx
            |--view.tsx
    |--mobile  
        |--Button
            |--index.tsx
            |--view.tsx
```

3. 自定义组件代码（下面以pc举例）
    - 快速集成已存在的组件, 要求已存在的组件必须有抛出值的事件如onChange事件存在，或者不参与表单值的组件那就没有任何要求
        - 修改 `index.tsx`   
            - 比如我想集成antd的Button组件  

                ```ts
                import { defineRfRenderComponent } from '@rf-render/antd'
                import { Button } from 'antd'
                export default defineRfRenderComponent(Button)
                ```
            - 假如我有个组件叫UserSelct,他抛出值的事件叫onValue

                ```ts
                import { defineRfRenderComponent } from '@rf-render/antd'
                import UserSelct from 'UserSelct'
                export default defineRfRenderComponent(UserSelct, {
                    propsMap: {
                        onChange: 'onValue'
                    }
                })
                ```
            - 如果我想支持mapKeys属性，比如UserSelct组件只给了id， 我还想返回整个对象，其中包含了name等字段, 那就需要自定义onMapKeysChange返回这个对象

              ```ts
                import { defineRfRenderComponent } from '@rf-render/antd'
                import UserSelct from 'UserSelct'
                export default defineRfRenderComponent(UserSelct, {
                    // 这里的val是changeConfig返回的值, rfrender里包含了当前表单的配置项
                    onMapKeysChange:(rfrender, ...val) => {
                        const userInfo  {}
                        // 在这里可以经过各种处理得到userInfo
                        return [userInfo]
                    },
                    propsMap: {
                        onChange: 'onValue'
                    }
                })
                ```
            - 完全自定义
                - 完全自定义只要记住两点
                    1. 插件类型是单纯的layout组件还是参与表单运算的组件 Layout组件
                    2. 调用props.onChange事件以修改表单值,如果要支持mapKeys属性请调用props.onMapKeysChange事件并传入一个数组，数组的值会和mapKeys定义的键顺序对应
                ```tsx
                import { FormItemBridgeProps } from '@rf-render/core'
                interface MyCustomerComponentProps {
                    prop1?: string
                    prop2: string
                }
                export function MyCustomerComponent(props: FormItemBridgeProps & MyCustomerComponentProps) { 
                    const { rfrender, prop1, prop2, onChange, onMapKeysChange } = props
                    const { item } = rfrender
                    // 如果是layout组件可以获取layout配置进行自定义布局, item是当前表单项的配置
                    const { layout = [] } = item
                    return <input placeholder={prop1} type={prop2} onInput={(e) => {
                        const value = e.target.value
                        onChange(value)
                        onMapKeysChange([`mapKeys-index0-${value}`, `mapKeys-index1-${value}`])
                    }}/>
                }
                ```
                具体可以查看`@rf-render/antd`包下面的plugin文件夹
                


4. 拓展widget及组件属性  
    - 修改`types/@rf-render/antd/index.d.ts`文件新增以下内容
    
        ```ts
        import '@rf-render/antd'
        import { AntdDefaultWidgetProps, AntdWidgetProps } from '@rf-render/antd'
        import { ButtonProps } from 'antd'
        import UserSelctProps from 'UserSelctProps'
        declare module '@rf-render/antd' {
            export interface WidgetProps extends AntdWidgetProps {
                // 在WidgetProps中新增一个widget类型Button, 并配置其属性
                Button: ButtonProps
                 // 在WidgetProps中新增一个widget类型UserSelct, 并配置其属性
                UserSelct: UserSelctProps
            }
            export interface DefaultWidgetProps extends AntdDefaultWidgetProps {}
        }
        ```

5. 创建插件导出
    - 新增文件 `rf-render-components/index.tsx`, 写入以下内容
        ```ts
        import { definePlugin } from '@rf-render/core'
        import { lazy } from 'react'
        /**
         * 这里我们只适配了pc，如果只支持pc则修改`${platform}`为 'pc' 即可
        */
        export const customerRfRenderPlugin = definePlugin([
            {
                name: 'Button',
                loader: (platform, fileName) => lazy(() => import(`./Button/${platform}/${fileName}.tsx`)),
            },
            {
                name: 'UserSelct',
                loader: (platform, fileName) => lazy(() => import(`./UserSelct/${platform}/${fileName}.tsx`)),
            },
        ])
        ```
6. 注册插件

```ts
import customerRfRenderPlugin from "./src/rf-render-components/index.tsx"
new RfRender({
  // ...其他配置
  plugins: [
    // ...其他插件
    ...antdRfRenderPlugin,
    ...customerRfRenderPlugin
  ],
})
```
